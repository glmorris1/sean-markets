# Sean Markets

Sean is a static clone of [TradingView](https://www.tradingview.com/) — rebranded as **Sean** — with a marketing homepage and a full charting workspace.

## Pages

- `index.html` — Landing page with market summary, ticker, and community ideas
- `chart.html` — Chart terminal with watchlist, candlesticks, indicators, and trading panel

## Run locally

```sh
python3 -m http.server 4173
```

Open [http://127.0.0.1:4173](http://127.0.0.1:4173).
